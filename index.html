1. ما هو HTML؟

HTML (لغة ترميز النصوص التشعبية) هي الهيكل الأساسي لأي موقع ويب. تُستخدم لتحديد العناصر مثل: العناوين، الفقرات، والصور.




    صفحتي الأولى


    
مرحباً في عالم الويب!

    
هذه أول فقرة لي في صفحة ويب.

2. ما هو CSS؟

CSS هو ما يجعل صفحتك جميلة! يُستخدم لتغيير الألوان، الخطوط، والتنسيق.

body {
    font-family: Arial;
    background-color: #f7f7f7;
    text-align: center;
}
h1 {
    color: #4169e1;
}
